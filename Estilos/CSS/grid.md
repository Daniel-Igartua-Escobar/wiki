# GRID

## Crear contenedor grid con las columnas personalizadas

```css
.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}
```

## Espacio entre los elementos

```css
.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 10px;
     /* column-gap: 10px; */
    /* row-gap: 15px; */
}
```
